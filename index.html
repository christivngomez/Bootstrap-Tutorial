Bootstrap Tutorial

Bootstrap is the most popular HTML, CSS, nad JS framework for developing responsive, mobile-first projects on the web.

Creating a Jumbotron

A jumbotron indicates a big box for calling extra attention to some special content or information. Inside a jumbotron you can put nearly any valid HTML, including other Bootstrap elements/classes.

Navigation Bars

A navigation bar is a navigation header that is placed at the top of the page. With Bootstrap, a navogation bar can extend or collapse, depending on the screen size.

Bootstrap Grid System

Bootstrap's grid system allows up to 12 columns across the page. If you do not want to use all 12 columns individually, you can group the columns together to create wider columns. Bootstrap's grid system is responsive, and the columns will re-arrange automatically depending on the creen size.
